<h1>
  This repository's purpose is to learn how to setup a new git repo
</h1>

<ol>
  <li>
    git init
  </li>
  <li>
    git add file.ext
  </li>
  <li>
    git commit -m "Message"
  </li>
  <li>
    git remote add origin url-to-github
  </li>
  <li>
    git push -u origin master
  </li>
</ol>

<ol>
  <li>
    Fork repository
  </li>
  <li>
    Clone the forked directory
  </li>
  <li>
    Do changes
  </li>
  <li>
    git add .
  </li>
  <li>
    git commit -m "message"
  </li>
  <li>
    git git push
  </li>
  <li>
    Pull request
  </li>
</ol>
